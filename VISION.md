# Payday Planner Vision

Payday Planner helps people decide what each pay should do before the money is spent.

It is a financial planning tool, not an accounting package.

## Principles

- Plan ahead rather than only looking backwards.
- Reduce repetitive work.
- Keep the interface simple.
- Automate recurring tasks, not occasional one-off tasks.
- Show only what still needs attention.
- Design mobile first.

## Design philosophy

The app is a digital version of post-it notes placed on simple financial planning boards.

Every new feature should make planning a future pay easier.
